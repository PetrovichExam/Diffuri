Данный метод нахождения численного решения называется методом Эйлера (или методом ломаных). Схемы, в которых значение функции явно выражается через уже найденные значения, называются явными, иначе – неявными. Таким образом, схема Эйлера является явной. Оценка погрешности для данного метода дает O(max(hi)), что предполагает малый шаг сетки для получения удовлетворительного решения.
На каждом из отрезков [xi, xi+1] полученное решение будет представлять собой отрезок прямой, проведенной через точку (xi, yi) с угловым коэффициентом f(xi, yi). Такая геометрическая интерпретация решения объясняет второе название метода (метод ломаных).
